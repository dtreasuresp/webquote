# 09 - AUDITORÍAS

Auditorías del sistema, reportes de calidad, gobernanza y controles.

## Contenido

- **audits-dir/** - Auditorías técnicas y de seguridad

## ¿Cuándo usarlo?

- Para revisar auditorías de seguridad o calidad
- Cuando necesites reporte de estado del proyecto
- Para entender políticas y controles

- Reportes de rendimiento
